Test Favorite Button (Giả Lập)

Nhấn nút dưới để test (giả lập mà không cần đăng nhập thực):

Cách Hoạt Động

  1. Khi user nhấn nút "Yêu thích"
  2. JavaScript gọi toggleFavorite(postId, button)
  3. Gửi AJAX POST đến FavoriteController.php
  4. Body: "action=add&post_id=1" (form-urlencoded)
  5. Controller kiểm tra action: 'add' hoặc 'remove'
  6. Thực hiện thêm/xóa vào database
  7. Trả về JSON: {success: true, message: "...message..."}
  8. JavaScript cập nhật UI và hiển thị notification

Các File Đã Sửa

  • assets/js/main.js - Cập nhật toggleFavorite function:
    • Gửi action='add' (nếu chưa favorite) hoặc action='remove' (nếu đã favorite)
    • Gửi form-urlencoded thay vì JSON
    • Format: 'action=add&post_id=1'
